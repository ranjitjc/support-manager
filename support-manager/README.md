# Welcome to ASP.NET 5

## Step #1: ASP.Net Project Creation
* Install dnx
    >dnvm install 1.0.0-rc1-final
* Create a project
    >yo aspnet
* Install Server Packages
    >dnu restore
* Install Client Packages
    >bower install    
* Build the Project
    >dnu build --quiet
* Run the Project
    >dnx web
